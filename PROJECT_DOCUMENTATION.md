# توثيق مشروع تطبيق تذكير شرب الماء

## نظرة عامة

تم إنشاء تطبيق Android بلغة **Kotlin** لإرسال إشعارات تذكيرية منتظمة للمستخدم لشرب الماء. التطبيق يستخدم أحدث تقنيات Android ويتبع أفضل الممارسات في تطوير التطبيقات.

## رابط المستودع

**GitHub Repository:** [https://github.com/bbggjah-commits/my-APK-abb](https://github.com/bbggjah-commits/my-APK-abb)

## البنية المعمارية

### المكونات الرئيسية

#### 1. MainActivity.kt
النشاط الرئيسي للتطبيق الذي يحتوي على:
- واجهة المستخدم الرئيسية
- إدارة أذونات الإشعارات لنظام Android 13 وما فوق
- التحكم في بدء وإيقاف التذكيرات
- التحقق من صحة المدخلات

#### 2. WaterReminderWorker.kt
فئة Worker المسؤولة عن:
- تنفيذ المهام في الخلفية باستخدام WorkManager
- إنشاء وإدارة قنوات الإشعارات
- عرض الإشعارات للمستخدم
- التعامل مع إصدارات Android المختلفة

#### 3. BootReceiver.kt
مستقبل البث (Broadcast Receiver) الذي:
- يعيد تشغيل التذكيرات تلقائياً بعد إعادة تشغيل الجهاز
- يحافظ على استمرارية الخدمة

### التقنيات المستخدمة

| التقنية | الغرض | الإصدار |
|---------|-------|---------|
| **Kotlin** | لغة البرمجة الأساسية | 1.8.0 |
| **Android Gradle Plugin** | بناء المشروع | 8.0.2 |
| **AndroidX** | مكتبات Android الحديثة | متنوع |
| **WorkManager** | جدولة المهام في الخلفية | 2.8.1 |
| **Material Components** | تصميم واجهة المستخدم | 1.9.0 |
| **ConstraintLayout** | تخطيط الواجهة | 2.1.4 |

## المميزات الرئيسية

### 1. إشعارات ذكية
- استخدام **WorkManager** لضمان موثوقية الإشعارات
- دعم جميع إصدارات Android من API 21 (Android 5.0) وما فوق
- إشعارات ذات أولوية عالية لضمان ظهورها

### 2. واجهة مستخدم حديثة
- تصميم **Material Design 3**
- واجهة عربية كاملة
- ألوان متناسقة مع موضوع الماء (أزرق)
- أيقونة قطرة ماء مخصصة

### 3. إدارة الأذونات
- طلب أذونات الإشعارات تلقائياً لنظام Android 13+
- معالجة رفض الأذونات بشكل صحيح
- رسائل توضيحية للمستخدم

### 4. مرونة في الإعدادات
- إمكانية تخصيص الفترة الزمنية بين التذكيرات
- القيمة الافتراضية: 60 دقيقة
- الحد الأدنى: دقيقة واحدة

### 5. الاستمرارية
- استمرار التذكيرات حتى بعد إعادة تشغيل الجهاز
- استخدام **BootReceiver** لإعادة الجدولة

## كيفية البناء والتشغيل

### المتطلبات
1. **Android Studio** Arctic Fox أو أحدث
2. **JDK** 8 أو أحدث
3. **Android SDK** مع API Level 33

### خطوات البناء

1. **استنساخ المستودع:**
```bash
git clone https://github.com/bbggjah-commits/my-APK-abb.git
cd my-APK-abb
```

2. **فتح المشروع في Android Studio:**
   - File → Open
   - اختر مجلد المشروع

3. **مزامنة Gradle:**
   - سيقوم Android Studio تلقائياً بتنزيل التبعيات

4. **تشغيل التطبيق:**
   - اضغط على زر Run (▶️)
   - اختر جهاز أو محاكي

### بناء APK

```bash
./gradlew assembleDebug
```
سيتم إنشاء ملف APK في: `app/build/outputs/apk/debug/app-debug.apk`
